# YTMetrics Desktop — тестовая сборка

## Что это
Обёртка твоего сайта в нативное окно (Electron). Бэкенд остаётся на Vercel как есть —
приложение просто открывает его в своём окне вместо браузера. Google-вход открывается
в обычном браузере и возвращается в приложение автоматически.

## Как запустить (на каждой ОС отдельно — Windows, macOS, Linux)

1. Установи Node.js (если ещё нет): https://nodejs.org (версия LTS)
2. Скопируй папку `electron-app` на компьютер
3. Открой терминал в этой папке и выполни:
   ```
   npm install
   npm start
   ```
4. Откроется окно приложения — проверь, что сайт загружается и вход через Google работает.

## Важно про Google-вход
Чтобы вход через Google сработал в приложении, в Google Cloud Console
(в том же проекте, где вы брали ключи для входа) НЕ нужно ничего менять —
мы используем уже существующий redirect_uri на Vercel. Приложение просто
подхватывает готовый токен через ссылку `ytmetrics://auth-callback`,
которую сервер теперь умеет обрабатывать (см. изменение в `index.js`).

## Что нужно сделать тебе на Vercel
Обнови файл `index.js` (бэкенд) на этот новый вариант — я обновил всего 1 строку
(добавил поддержку `ytmetrics://`). Больше ничего менять не нужно.

## Дальше (не сейчас, отдельным шагом)
Когда тестовая сборка подтвердит, что всё работает — соберём подписанные
установщики (.exe / .dmg / .AppImage) через electron-builder для продажи.
Для macOS подписи понадобится Apple Developer аккаунт ($99/год),
для Windows — код-签 сертификат (можно и без него, но будет предупреждение
"неизвестный издатель" при установке).
